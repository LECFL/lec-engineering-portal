# LEC Engineering Portal

A modern internal engineering portal for Lubavitch Educational Center's software ecosystem.

This first version is a dependency-free static site designed for quick Netlify deployment. It showcases LEC's applications, services, architecture, technologies, and roadmap using the master cross-repo context as the source of truth.

## Sections

- Home / platform overview
- Featured systems
- Project catalog
- Architecture map
- Technology explorer
- Roadmap
- About LEC Engineering

## Netlify

The site is static and deploys from the repository root.

- Publish directory: `.`
- Build command: none
